חיבור ל-Firebase

פעולה חד-פעמית — אחר כך הכל אוטומטי

1
פתח console.firebase.google.com
התחבר עם חשבון Google שלך
2
צור פרויקט חדש
שם כלשהו (לדוגמה: "quote-app") ← המשך ← המשך ← צור
3
הפעל Firestore
בתפריט שמאל: Build ← Firestore Database ← Create database ← Start in test mode ← המשך
4
קבל את הגדרות הפרויקט
⚙️ Project settings ← General ← Your apps ← לחץ </> ← Register app ← העתק את אובייקט firebaseConfig
5
פתח את קובץ config.js בנוטפד
החלף את כל ה-REPLACE_ME בערכים שהעתקת ← שמור
6
רענן את הדף
F5 — האפליקציה תתחבר ותיפתח
